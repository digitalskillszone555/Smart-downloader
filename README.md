# ⚡ Smart Downloader Pro

A powerful, high-performance media extractor for modern social platforms. Built for speed, quality, and simplicity.

## 🚀 Features
- **True 4K & HD Video:** Downloads up to 2160p resolution with high-fidelity audio merging.
- **Pure Audio Extraction:** Converts videos to high-quality MP3 (up to 320kbps) for music players.
- **HQ Image Downloader:** Saves uncorrupted photos from Instagram, X, and more.
- **Auto-Updating Engine:** Automatically keeps `yt-dlp` updated for maximum compatibility.
- **Premium UI:** Modern Glassmorphism dark theme with real-time progress tracking.

## 🛠️ Requirements
- **Python 3.10+**
- **FFmpeg:** Required for 4K video merging and MP3 conversion.
  - *Windows:* `choco install ffmpeg`
  - *Mac:* `brew install ffmpeg`
  - *Linux:* `sudo apt install ffmpeg`

## 📦 Installation & Usage
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```

## 📂 Downloads
All media is saved to your local `Downloads/SmartDownloader` folder.
