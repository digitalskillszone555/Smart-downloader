# Privacy Policy

**Smart Downloader Pro** values your privacy. This application is designed to operate with a **privacy-first** philosophy.

## Data Collection
- **Local Operation:** This application runs entirely on your local machine.
- **No Remote Storage:** We do not collect, store, or transmit any user data, download history, or IP addresses to external servers.
- **No Third-Party Tracking:** There are no integrated tracking scripts or analytics within the core application logic.

## Media Processing
- All media processing (video merging, audio extraction) is performed locally using your machine's hardware and the `ffmpeg` library.
- No media is ever uploaded to a cloud server for conversion.

## Your Responsibility
- By using this tool, you agree that any media downloaded is for personal, non-commercial use only.
- You are responsible for ensuring that your use of this software complies with the terms of service of the platforms you access.
